Switching

cec-active-source announces this box as the active CEC source, switching every display on the bus to this input — it is the "switch input" primitive; there is no separate switch-input command. Power actions target one device by logical address (from the Topology scan above).

Input name

The OSD device name this box announces on the CEC bus — the input label TVs/AVRs display. Currently {{ osd_effective }} ({{ osd_source }}). Empty = use the hostname. Max 13 ASCII characters (libcec limit). Saving writes [cec].osd_name in config.toml; restart tv-shell-input (Processes page) to announce the new name. Note: other CEC-capable apps that grab the adapter (e.g. Plex HTPC with CEC enabled) announce their own name — keep CEC disabled in apps so the daemon stays the single CEC owner.
